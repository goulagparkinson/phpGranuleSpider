PHP Granule Spider
======================

A script to assist browsing filesystem and matching Earth Observation satellite dataset.

Requirements

 * PHP >= 5.3
 * tree unix command line >= 1.6 (with -X XML output opt flag)


Usage
-----

See `phpGranuleSpider_sample.php` for examples.
